# Hospital Appointment System Frontend

React UI for the Hospital Appointment System backend API.

## Setup

1. Start backend from the backend folder:

```bash
cd ../hospital_appointment_system_backend
npm install
npm start
```

2. Start frontend:

```bash
cd ../hospital_appointment_system_frontend
npm install
npm run dev
```

## Environment Variables

- `VITE_API_BASE_URL` defaults to `http://localhost:5000/api/v1` in `.env`
